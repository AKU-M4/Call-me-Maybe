Core Concepts to Learn
1. How LLMs Work (Tokenization → Logits → Token Selection)
This is the foundation of the whole project. You need to understand:

How text gets broken into tokens (BPE/SentencePiece)
What logits are and how they represent probability distributions
How autoregressive generation works (token by token)

Resources:

Andrej Karpathy — "Let's build the GPT Tokenizer" (YouTube) — best hands-on intro to tokenization
3Blue1Brown — "But what is a GPT?" (YouTube) — visual intuition for the full pipeline
Hugging Face NLP Course — Tokenizers chapter — practical and clear


2. Constrained Decoding (the heart of this project)
This is the hardest and most important concept. You need to understand how to intercept the logits at each generation step and mask out tokens that would violate your JSON schema.
Resources:

Lilian Weng — "Constrained Decoding" — excellent conceptual overview
Outlines library source code — even though you can't use it, reading how it works is gold
Blog: "Guaranteed Valid JSON from LLMs" — practical walkthrough of the idea
"Efficient Guided Generation for LLMs" (arxiv) — the academic paper behind the technique

The mental model: at each step, build a set of valid next tokens based on where you are in the JSON structure, set everything else to -inf, then let the model pick.

Key Implementation Insight
The core loop you'll build looks like this:
for each generation step:
    1. get logits from the model (using llm_sdk)
    2. figure out which tokens are valid *right now* based on JSON state
    3. set invalid token logits to -inf
    4. pick the highest remaining logit
    5. append that token and repeat
The trickiest part is step 2 — you need a state machine or parser that tracks where you are in the JSON structure (inside a string? inside a number? expecting a key?). Study how the outlines library builds its FSM (finite state machine) for this — even if you can't use it, the approach is exactly what you need to replicate.
Good luck — this is a genuinely interesting project once the pieces click together!
